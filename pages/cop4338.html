COP4338: Programming III

Description

Class which dove deep into Java, including multithreading, sychronization, and networking.

Final Project

Develop a memory game which gets two identical images and forms a pair. The game uses a N by N table of cards. Using coordinates, each player clicks on a card and if both cards match, the player receives a point. If the cards so not match, the next player will play. Board is a 6x6 matrix of cards, built as a 2-dimensional linked list.

Code

This game was designed to be played either single player or multiplayer through p2p socket TCP connections.
