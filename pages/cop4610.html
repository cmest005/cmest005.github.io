Sample work

This project introduced process management. We were instructed to create programs which implement sychronization amongst threads using the Pthread library.
The first part was a program which I created a function named SimpleThread, which spawned a number of threads that allow the use of a global variable to do some calculations. Then I used mutex locks to synchronize threads so that each thread incremented the global variable by one and each thread displayed the same final value.
