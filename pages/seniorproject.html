Senior Project

Description

Our project was to start version 1.0 of a results section for the DCU (Drinker's Checkup) website. The DCU is a virtual health counselor, which interacts with a user who may want to learn more about their alcohol issues and seek help for them. The results of the diagnoses from the questionnaire pre-project were just a text-based output. My partner and I were in charge of developing an interactive results section that engaged the user with the website in learning about their results. Our pages would use HTML, CSS, and JavaScript to design tables and graphs to display results.
This was a very interesting final project. The system is set up using 2 main systems to upload and control content to the site: Survey Editor, State Machine. The Survey Editor is where all the content is created and uploaded. This includes buttons, html pages, and tranactions with the counselor based on user input. Once the content is created, the State Macine is set up to display the content in an order determined by the end-user's input.
