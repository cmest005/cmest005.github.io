Description

Our Database class was taught using SQL and testing using MySQL through MySQL Workbench and phpMyAdmin. The intended outcome was to expose students to information systems and familiarize us with database systems and architectures. Some of the most interesting topics covered were: different joins, design schemas, create queries in SQL, relational algebra, and relational calculus.
