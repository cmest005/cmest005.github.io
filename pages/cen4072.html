CEN4072: Software Testing

Description

Our semester-long project was to do Unit, Subsystem, and System testing on a pre-existing web application. The web app was named Schedule Maker, and it was an online resource for students which allowed them to search for classes, create schedules, and register for the classes. It was a java based program with the User Interface written in JSP, and the database in SQL.

As part of the System team, I used IBM's Rational Function Tester (RFT) to use custom verification points in order to test that the system was working as intended. For every Functional Requirement we wrote 3 Sunny Day, and 3 Rainy Day test cases. With the app running on a Tomcat server through Eclipse, RFT could be scripted to run test cases using Java. Here is an example of a Sunny and Rainy day test case for the Login Function.
